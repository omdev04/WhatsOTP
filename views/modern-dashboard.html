Your API Token

Use this token to authenticate API requests from external tools (like Postman or Thunder Client)

How to use your API token:

  1. Add the x-api-token header to your API requests
  2. Include your token value as shown in the example below
{
  "headers": {
    "Content-Type": "application/json",
    "x-api-token": "your-token-here"
  }
}

Simple Integration

Send OTP

// Send OTP
fetch('/api/otp/send', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ phone: '919876543210' })
})
.then(response => response.json())
.then(data => console.log(data));

Verify OTP

// Verify OTP
fetch('/api/otp/verify', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ 
    phone: '919876543210',
    otp: '123456'
  })
})
.then(response => response.json())
.then(data => console.log(data));
