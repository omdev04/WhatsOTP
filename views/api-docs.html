WhatsOTP API Documentation

Welcome to the WhatsOTP API documentation. This guide will help you integrate WhatsApp-based OTP authentication into your applications with zero SMS costs.

WhatsOTP uses the WhatsApp platform to deliver OTPs, making it a cost-effective alternative to traditional SMS-based authentication.

Base URL

All API URLs referenced in this documentation have the following base:

http://yourserver.com/api

For local development, use: http://localhost:3000/api

Authentication

Authentication Required

All API endpoints require authentication. You can authenticate using session-based authentication when logged in through the web interface.

For API integrations, you need to be logged in to the WhatsOTP dashboard. The system will use your session cookie for authentication.

curl -X POST \
  http://localhost:3000/api/otp/send \
  -H 'Content-Type: application/json' \
  -d '"phone": "911234567890"' \
  --cookie "connect.sid=your-session-cookie"

Error Handling

WhatsOTP uses conventional HTTP response codes to indicate the success or failure of an API request:

  • 2xx range indicates success
  • 4xx range indicates an error in your request
  • 5xx range indicates a server error

All responses follow this consistent structure:

{
  "success": true|false,
  "message": "A human-readable message explaining the result",
  // Additional data depending on the endpoint
}

Send OTP

POST /otp/send

Sends a one-time password to a phone number via WhatsApp. The OTP is valid for 5 minutes.

Request Parameters

Parameter Type Description
phone * String Phone number with country code, without the '+' sign (e.g., "917666518073")

Example Request

{
  "phone": "911234567890"
}

Responses

Success (200 OK):

{
  "success": true,
  "message": "OTP sent successfully"
}

Error (400 Bad Request):

{
  "success": false,
  "message": "Phone number is required"
}

Error (500 Server Error):

{
  "success": false,
  "message": "Failed to send OTP"
}

Verify OTP

POST /otp/verify

Verifies a one-time password that was sent to a phone number.

Request Parameters

Parameter Type Description
phone * String Phone number with country code, without the '+' sign (e.g., "911234567890")
otp * String The 6-digit OTP received via WhatsApp

Example Request

{
  "phone": "911234567890",
  "otp": "123456"
}

Responses

Success (200 OK):

{
  "success": true,
  "message": "OTP verified successfully"
}

Error (400 Bad Request):

{
  "success": false,
  "message": "Invalid or expired OTP"
}

Code Examples

JavaScript / Node.js

const axios = require('axios');

// Your API token
const API_TOKEN = 'your-api-token-here';

// Function to send OTP
async function sendOTP(phoneNumber) {
  try {
    const response = await axios.post('http://localhost:3000/api/otp/send', {
      phone: phoneNumber
    }, {
      headers: {
        'Content-Type': 'application/json',
        'x-api-token': API_TOKEN
      }
    });
    
    console.log(response.data);
    return response.data;
  } catch (error) {
    console.error('Error sending OTP:', error.response?.data || error.message);
    throw error;
  }
}

// Function to verify OTP
async function verifyOTP(phoneNumber, otpCode) {
  try {
    const response = await axios.post('http://localhost:3000/api/otp/verify', {
      phone: phoneNumber,
      otp: otpCode
    }, {
      headers: {
        'Content-Type': 'application/json',
        'x-api-token': API_TOKEN
      }
    });
    
    console.log(response.data);
    return response.data;
  } catch (error) {
    console.error('Error verifying OTP:', error.response?.data || error.message);
    throw error;
  }
}

Python

import requests

# Your API token
API_TOKEN = "your-api-token-here"
BASE_URL = "http://localhost:3000/api"

# Function to send OTP
def send_otp(phone_number):
    try:
        response = requests.post(
            f"{BASE_URL}/otp/send",
            json={"phone": phone_number},
            headers={
                "Content-Type": "application/json",
                "x-api-token": API_TOKEN
            }
        )
        response.raise_for_status()
        return response.json()
    except requests.exceptions.RequestException as e:
        print(f"Error sending OTP: {e}")
        return None

# Function to verify OTP
def verify_otp(phone_number, otp_code):
    try:
        response = requests.post(
            f"{BASE_URL}/otp/verify",
            json={"phone": phone_number, "otp": otp_code},
            headers={
                "Content-Type": "application/json",
                "x-api-token": API_TOKEN
            }
        )
        response.raise_for_status()
        return response.json()
    except requests.exceptions.RequestException as e:
        print(f"Error verifying OTP: {e}")
        return None
